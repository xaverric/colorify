# Colorify
Color palette generator

* Simple color palette generator
* Returns colors in RGB integer values

## Usage
```
Colorify colorify = new Colorify();
Palette palette = colorify.getPalette();

for (Color color : palette.getPalette()){
  color.getRed();
  color.getGreen();
  color.getBlue();
}

```
